# CSV取り込みガイド

データ入力画面から、レセコン（Recepty等）のCSVをアップロードして月次実績に取り込めます。

## 対応形式

- 拡張子: `.csv`
- 文字コード: UTF-8（BOM付きにも対応）
- 1行目: ヘッダー（列名）
- 2行目以降: データ行

## 列名マッピング

以下の列名（ヘッダー）が自動でマッピングされます。複数の別名に対応しています。

| 取り込み先 | 対応する列名の例 |
|-----------|-----------------|
| 処方箋受付件数 | 処方箋受付件数、処方箋件数、処方箋数、受付件数 |
| 在宅患者訪問薬剤管理指導件数 | 在宅患者訪問、在宅訪問、在宅件数 |
| 服薬フォローアップ件数 | 服薬フォローアップ件数、服薬フォロー |
| かかりつけ薬剤師算定件数 | かかりつけ算定 |
| 麻薬調剤件数 | 麻薬調剤件数、麻薬件数 |
| 重複投薬・相互作用等防止加算件数 | 重複防止加算 |
| マイナ保険証確認割合 | マイナ保険証確認割合、マイナ確認率、保険証確認割合 |
| 電子処方箋受付件数 | 電子処方箋、電子処方箋受付件数 |
| 時間外・休日対応件数 | 時間外対応、休日対応 |

## サンプルCSV

```csv
日付,処方箋受付件数,在宅患者訪問,電子処方箋,マイナ保険証確認割合
2024-03-01,45,2,10,85
2024-03-02,52,1,12,82
2024-03-03,38,0,8,90
```

- 複数行ある場合、数値は合算されます
- 日付・患者名など個人情報の列は無視されます

## 制限

- ファイルサイズ: 10MB以下
- 行数: 10万行以下
